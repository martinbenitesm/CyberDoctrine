CyberDoctrine es un repositorio personal de conocimientos sobre ciberseguridad centrado en la teoría, la investigación, la documentación técnica y el material de estudio en múltiples ámbitos de la ciberseguridad.

---
## Asignaturas

📗 Arquitectura y Fundamentos
- Redes de Datos y Comunicaciones
- Sistemas Operativos
- Arquitectura de Windows (Windows Internals)
- Desarrollo y Programación Técnica

📕 Seguridad Ofensiva y Análisis
- Criptografía Aplicada
- Análisis de Malware
- Ingeniería Inversa
- Seguridad en Aplicaciones Web
- Inteligencia de Fuentes Abiertas (OSINT)

📘 Seguridad Defensiva y Operaciones
- Respuesta a Incidentes y Análisis Forense Digital (DFIR)
- Seguridad en Infraestructura Cloud
- Seguridad en Entornos de Active Directory
- Seguridad en Dispositivos Móviles
- Seguridad en Redes Inalámbricas

---

## Aspiración

Este repositorio funciona como una biblioteca centralizada de doctrina de ciberseguridad para:

- Investigación y consulta académica: Facilitar el acceso a fuentes de información técnica.
- Gestión de documentación técnica: Estandarizar el registro de procesos y normativas.
- Desarrollo de capacidades en Blue Team y Red Team: Profundizar en metodologías ofensivas y defensivas.
- Fomento del aprendizaje continuo: Servir como base para la actualización profesional permanente.

Cada directorio integra una compilación exhaustiva de documentación técnica, incluyendo archivos PDF, notas de campo, diagramas de arquitectura y referencias bibliográficas vinculadas a su dominio específico.
